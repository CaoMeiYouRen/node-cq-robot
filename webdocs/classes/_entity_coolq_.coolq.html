getRecord

  • getRecord(file: string, out_format: string, full_path: boolean): Promise<string>
    • Defined in entity/CoolQ.ts:1432

    接收语音。 其实并不是真的获取语音，而是转换语音到指定的格式，然后返回语音文件名（data\record 目录下）。注意，要使用此接口，需要安装 酷Q 的 语音组件。

    author

    CaoMeiYouRen

    date

    2019-07-13

    memberof

    CoolQ

    Parameters

    • file: string

      收到的语音文件名（CQ 码的 file 参数），如 0B38145AA44505000B38145AA4450500.silk

    • out_format: string

      要转换到的格式，目前支持 mp3、amr、wma、m4a、spx、ogg、wav、flac

    • full_path: boolean

      是否返回文件的绝对路径（Windows 环境下建议使用，Docker 中不建议）

    Returns Promise<string>

    转换后的语音文件名或路径，如 0B38145AA44505000B38145AA4450500.mp3，如果开启了 full_path，则如 C:\Apps\CoolQ\data\record\0B38145AA44505000B38145AA4450500.mp3
